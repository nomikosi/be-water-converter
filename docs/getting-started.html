General

This is a fun-made project. I always wanted to create an IntelliJ plugin. If you find any issues, feel free to create a PR at https://github.com/nomikosi/be-water-converter or message me directly.

If you have any other ideas about improvements, feel free to share those at nomikosi.consulting at gmail.com.

1. Install the plugin

Open your JetBrains IDE, go to Settings → Plugins → Marketplace, search for Be Water Converter, and click Install. Restart the IDE when prompted.

2. Open the tool window

Click the Be Water tab on the right side bar, or go to Tools → Be Water Converter.

3. Load your data

Paste text into the input editor, or load a file using one of these methods:

  • Drag and drop a file onto the input editor
  • Open a file from the toolbar button or with Ctrl+Shift+O

The source format is auto-detected from the file extension.

4. Convert

Select the source format in the From dropdown and the target format in the To dropdown, then click Convert or press Ctrl+Enter.

Keyboard shortcuts

Shortcuts are active while focus is inside the Be Water tool window, including inside the input and output editors.

  • Ctrl+Enter converts input to the selected output format
  • Alt+Shift+L formats or pretty-prints the input
  • Alt+Shift+C copies output to the clipboard
  • Ctrl+Shift+O opens a file as input
  • Ctrl+Shift+S saves the current output to a file

Conversion-specific options

Some output formats show an options bar below the toolbar:

  • CSV choose between FLAT_FIRST (expands the first array only) and CROSS_JOIN (full Cartesian product). In CROSS_JOIN mode, adjust the Row warning threshold to control when the confirmation dialog appears.
  • Java POJO toggle Lombok annotations to add @Data, @NoArgsConstructor, and @AllArgsConstructor to generated classes.

Swap behavior

The swap button exchanges input and output when the output format can also be parsed as an input format. Generated Java POJO output is output-only, so it cannot be swapped back into the input editor as a source format.
